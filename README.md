## Installation

1. Clone this repository:
   ```sh
   git clone https://github.com/yourusername/craftybay.git
   ```
2. Navigate to the project directory:
   ```sh
   cd craftybay
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the application:
   ```sh
   flutter run
   ```
